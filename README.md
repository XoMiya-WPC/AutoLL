# AutoLL

<h1>Intro</h1>
A Simple Plugin for EXILED that adds Lobby Config.
Adds the Smod2 Config allowing lobbies to be locked unless a certain number of players have joined.

<h1>Config</h1>

| Config  | Type | Value |
| ------------- | ------------- | ------------- |
| IsEnabled  | Boolean  | true  |
| MinPlayersToStart  | Integer  | 4  |
